# My Empire: Kingpin — Grow Room & Strain Reference

## How Rooms Work

Each room is a building you buy with **dirty cash**. When you upgrade a room, you unlock
an additional strain slot — all slots grow simultaneously. Max 4 slots per room (3 upgrades).

Each slot consumes **1 seed** per grow cycle and auto-replants if seeds are in stock.

## Economy Notes

- **Dealer price** = each strain's `price/unit` × (1 - dealer cut%)
- **Street sell** = 70% of average price across all active slots
- More upgrade levels → higher average price → dealers earn more per unit sold
- Seeds cost $20/each (dirty cash). 1 seed consumed per slot per grow cycle.
- Auto-replant triggers on harvest if seed stock > 0
