### How to Poll for Tasks (run in Claude Code):
```bash
curl -s "http://10.0.1.147:5100/tasks/poll?workerId=worker-aipc2-001&domains=general,code,dev"
```

### How to Submit Results:
```bash
curl -s -X POST "http://10.0.1.147:5100/tasks/TASK_ID/result" \
  -H "Content-Type: application/json" \
  -d '{"workerId":"worker-aipc2-001","success":true,"outputs":{"response":"YOUR_RESULT"}}'
```

### How to Send Heartbeat:
```bash
curl -s -X POST "http://10.0.1.147:5100/health/heartbeat" \
  -H "Content-Type: application/json" \
  -d '{"workerId":"worker-aipc2-001","status":"OK","version":"1.0.0"}'
```
